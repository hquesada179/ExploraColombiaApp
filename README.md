# ExploraColombiaApp

Aplicación móvil Android desarrollada en **Kotlin** enfocada en mostrar información, exploración o interacción relacionada con Colombia desde una interfaz moderna en Android Studio.

## Descripción

**ExploraColombiaApp** es un proyecto Android creado en Android Studio.  
Su propósito es servir como base para una app móvil desarrollada en Kotlin, organizada con Gradle y estructura estándar de Android.

Este repositorio contiene el código fuente principal del proyecto, los archivos de configuración de compilación y la estructura base para continuar su desarrollo.

## Características

- Proyecto Android desarrollado en Kotlin
- Estructura organizada con Gradle
- Base para futuras pantallas, funcionalidades y navegación
- Repositorio versionado con Git y alojado en GitHub

## Tecnologías utilizadas

- **Kotlin**
- **Android Studio**
- **Gradle**
- **Git y GitHub**
